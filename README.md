## Getting Started

Python3 is required to install and run this program.

### Prerequisites

Run in a terminal:
  ```sh
  pip3 install matplotlib numpy
  ```

### Installation

Clone the repo:
  ```sh
  git clone https://github.com/sapblatt11/metaGenePlot.git
  ```
